# Getting Started

Welcome to @toneflix/money! This guide will help you get started with formatting currency, converting between currencies, and performing monetary calculations.

## What is @toneflix/money?

@toneflix/money is a zero runtime dependency library for working with money and currencies in JavaScript and TypeScript. It provides:

- **Currency Formatting**: Display amounts in any currency with proper symbols and formatting
- **Currency Conversion**: Convert between currencies using live exchange rates
- **Mathematical Operations**: Perform calculations with proper precision
- **TypeScript Support**: Full type safety included

## Why @toneflix/money?

- ✅ **Zero Runtime Dependencies** - Lightweight and fast
- ✅ **Type Safe** - Written in TypeScript with full type definitions
- ✅ **Easy to Use** - Intuitive API with method chaining
- ✅ **Well Tested** - Comprehensive test coverage
- ✅ **Production Ready** - Used in real-world applications
